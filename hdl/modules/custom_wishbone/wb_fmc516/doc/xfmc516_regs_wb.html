3. Register description

3.1. Control register

HW prefix: fmc516_fmc_ctl
HW address: 0x0
C prefix: FMC_CTL
C offset: 0x0
31 30 29 28 27 26 25 24
RESERVED[27:20]
23 22 21 20 19 18 17 16
RESERVED[19:12]
15 14 13 12 11 10 9 8
RESERVED[11:4]
7 6 5 4 3 2 1 0
RESERVED[3:0] CLK_SEL LED_1 LED_0 TEST_DATA_EN
  • TEST_DATA_EN [read/write]: Enable test data
    Write the address counter value instead of ADC data to Wishbone Streaming Interface
  • LED_0 [read/write]: Manual LED 0
    Manual control of the front panel LED 0
  • LED_1 [read/write]: Manual LED 1
    Manual control of the front panel LED 1
  • CLK_SEL [read/write]: Internal/External clock distribution selection
    Select between internal VCXO or external reference clock
    write 0: internal reference
    write 1: external reference
  • RESERVED [read/write]: Reserved
    Ignore on write, read as 0's

7 6 5 4 3 2 1 0
RESERVED[4:0] PRST PWR_GOOD MMCM_LOCKED
  • MMCM_LOCKED [read-only]: MMCM locked status
    ADC Clock Chain MMCM locked status.
    read 0: not locked
    read 1: locked
  • PWR_GOOD [read-only]: FMC power good status
    FMC power good status.
    read 0: power not good
    read 1: power good
  • PRST [read-only]: FMC board present status
    FMC board present status.
    0: not present
    1: present
  • RESERVED [read-only]: Reserved
    Ignore on write, read as 0's

3.4. Global ADC Status register

C prefix: ADC_STA
C offset: 0xc

15 14 13 12 11 10 9 8
- - - - DATA_CHAINS[3:0]
7 6 5 4 3 2 1 0
- - - - CLK_CHAINS[3:0]
  • CLK_CHAINS [read-only]: FMC ADC clock chains
    FMC ADC used clock chains.
    read 1: clock chain is used
    read 0: clock chain is not used
  • DATA_CHAINS [read-only]: FMC ADC Data chains
    FMC ADC used clock chains.
    read 1: data chain is used
    read 0: data chain is not used
  • ADC_PKT_SIZE [read-only]: FMC ADC packet size
    Size of the packet sent to the Wishbone Streaming Interface.
    This will appear as cycles of clock cycles on
    the wishbone interface
